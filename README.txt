Тестовое задание

Есть массив с double числами. Все числа равны, кроме одного. Попробуйте найти!
Гарантируется, что массив содержит не менее 3 чисел.
Входной массив может быть очень большим, так что подумайте о производительности.

Входной массив: 4, 4, 4, 4, 1, 4, 4
Результат: 1

Входной массив: 0, 0.02, 0, 0, 0, 0, 0
Результат: 0.02

1. Проверяю на наличие 2х разных чисел. Исходим из того что отличное число в массиве только 1
2. Тест на массив 100 млн чисел - метод проходит за время от 600 до 2500 мс, на 200 млн - переполнение heap )) 32 Гб ОЗУ
3. Нормализацию не проводил, т.к. нет в условии.
